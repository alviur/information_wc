How to execute integration code:

1. Run generate_alfa_b.m
2. Run generateKernel4TF.m
3. Using the files generated in 1, 2, and the path to Foster's dataset run integration.py (remember to include in the same folder utils.py)


NOTE: This code use Tensorflow version 1.13.1  

The integration is hardcoded for 3 scale an4 4 orientation wavelets
